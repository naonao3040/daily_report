### 学んだこと
- チャネルの作成 `make(chan string)`
- チャネルへの値の設定と取り出し
```
ch := make(chan string)
ch <- 'hoge'
fmt.Println(<-ch)
```
- ゴールーチンで処理を待つ時に`sync.WaitGroup`を利用することで全てのゴールーチンの処理が完了するまで待機できる。
```
var wg sync.WaitGrop

go func() {
  wg.Add(1)
  wg.Done()
}

go func() {
  wg.Add(1)
  wg.Done()
}
wg.Wait()
```
